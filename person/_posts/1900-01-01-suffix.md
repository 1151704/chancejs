---
title: suffix
signature: |
    chance.suffix()
    chance.suffix({full: true})
---

Generate a random name suffix

{% highlight js %}
chance.suffix();
=> 'Sr.'
{% endhighlight %}

By default, returns the shorter version.

Optionally get back the full version.

{% highlight js %}
chance.suffix({ full: true });
=> 'Juris Doctor'
{% endhighlight %}

*To maintain legacy support, this also responds to chance.name_suffix().*
