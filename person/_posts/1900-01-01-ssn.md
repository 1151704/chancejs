---
title: ssn
signature: |
    chance.ssn()
---

Generate a random social security number.

{% highlight js %}
chance.ssn();
=> '411-90-0070'
{% endhighlight %}

Optionally provide option of getting only the last four

{% highlight js %}
chance.ssn({ ssnFour: true });
=> '2938'
{% endhighlight %}

Optionally specify dashes be removed

{% highlight js %}
chance.ssn({ dashes: false });
=> '293839295'
{% endhighlight %}
