---
title: exp
signature: |
    chance.exp()
    chance.exp({raw: true})
---

Generate a random credit card expiration.

{% highlight js %}
  chance.exp();
  => '10/2020'
{% endhighlight %}

Optionally specify that a raw object be returned rather than a string

{% highlight js %}
  chance.exp({raw: true});
  => {month: '11', year: '2017'}
{% endhighlight %}
