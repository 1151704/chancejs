---
title: vat
signature: |
    chance.vat()
    chance.vat({country: 'it'})
---

Generate a random VAT id.

{% highlight js %}
chance.vat();
=> '00073160251'
{% endhighlight %}

Defaults to Italian VAT id -- Partita IVA). Optionally specify a country.

{% highlight js %}
chance.vat({country: 'it'});
=> '09516210367'
{% endhighlight %}

Note, currently support for country is limited to: `'it`.
