title: cli
---

To use Chance from the command line, install it globally with:

{% highlight bash %}
npm install -g chance
{% endhighlight %}

Then invoke any generator by name, like so:

{% highlight bash %}
$ chance name --prefix
Dr. Georgia Sanchez

$ chance latitude --min 12.34 --max 56.78
22.01836
{% endhighlight %}
