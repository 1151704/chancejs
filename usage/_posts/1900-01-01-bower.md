title: bower
---

It can also be used with [Bower](http://bower.io)

{% highlight bash %}
bower install chance
{% endhighlight %}

then in the HTML of your app:

{% highlight html %}
<!-- Load Chance -->
<script type="text/javascript" src="components/chance/chance.min.js"></script>
<script>
  // Use Chance
  alert(chance.string());
</script>
{% endhighlight %}
