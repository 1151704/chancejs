title: component
---

It can also be used with [Component](http://component.io)

{% highlight bash %}
component install chancejs/chancejs
{% endhighlight %}

then in the HTML of your app:

{% highlight html %}
<!-- Load Chance -->
<script type="text/javascript" src="components/chancejs-chancejs/chance.js"></script>
<script>
  // Use Chance
  alert(chance.string());
</script>
{% endhighlight %}
