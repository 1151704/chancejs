---
title: address
signature: |
    chance.address()
    chance.address({short_suffix: true})
---

Generate a random street address

{% highlight js %}
chance.address();
=> '5447 Bazpe Lane'
{% endhighlight %}

Optionally mandate that it return a short suffix

{% highlight js %}
chance.address({short_suffix: true});
=> '536 Baner Rd'
{% endhighlight %}
