---
title: province
signature: |
    chance.province()
    chance.province({full: true})
---

Return a random province.

{% highlight js %}
  chance.province();
  => 'ON'
{% endhighlight %}

By default, returns only the 2 letter abbreviation for province.

Optionally specify that it ought to return a full province name.

{% highlight js %}
  chance.province({full: true});
  => 'Nova Scotia'
{% endhighlight %}
