---
title: state
signature: |
    chance.state()
    chance.state({full: true})
---

Return a random state.

{% highlight js %}
  chance.state();
  => 'AK'
{% endhighlight %}

By default, returns only the 2 letter abbreviation for state.

Optionally specify that it ought to return a full state name.

{% highlight js %}
  chance.state({full: true});
  => 'Florida'
{% endhighlight %}
