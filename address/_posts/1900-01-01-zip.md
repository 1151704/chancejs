---
title: zip
signature: |
    chance.zip()
    chance.zip({plusfour: true})
---

Generate a random (U.S.) zip code.

{% highlight js %}
  chance.zip();
  => '90210'
{% endhighlight %}

Can optionally specify that it ought to return a [Zip+4][zip+4]:

{% highlight js %}
  chance.zip({plusfour: true});
  => '01035-1838'
{% endhighlight %}

[zip+4]: http://vq.io/19rzsve
