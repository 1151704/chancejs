---
title: timezone
signature: |
    chance.timezone()
---

Return a random timezone

{% highlight js %}
chance.timezone();
=> {  
      "name": "India Standard Time",
      "abbr": "IST",
      "offset": 5.5,
      "isdst": false,
      "text": "(UTC+05:30) Chennai, Kolkata, Mumbai, New Delhi",
      "utc": ["Asia/Calcutta"]
  }
{% endhighlight %}
