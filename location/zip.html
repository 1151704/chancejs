    // usage
chance.zip()
chance.zip({plusfour: true})

    Generate a random (U.S.) zip code.

    chance.zip();
=> '90210'

    Can optionally specify that it ought to return a Zip+4:

    chance.zip({plusfour: true});
=> '01035-1838'
